## Scoring Algorithm

```python
def calculate_score(solar, wind, elevation):
    solar_score    = normalize(solar, max=8.0)  * 0.50
    wind_score     = normalize(wind,  max=15.0) * 0.30
    elev_score     = normalize(elev,  max=2000) * 0.20
    return round((solar_score + wind_score + elev_score) * 100)
```

No ML training required â€” production-correct physics-based approach.
